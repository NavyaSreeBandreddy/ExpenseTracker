# 💰 Expense Tracker (Java Swing)

## 📌 Overview

A simple and user-friendly **Expense Tracker application** built using Java Swing.
It allows users to record daily expenses, view them in a table, and analyze spending using charts.

---

## 🚀 Features

* ➕ Add Expense (Title, Amount, Category, Date)
* 📋 View all expenses in a table
* ❌ Delete selected expense
* 💰 Total expense calculation
* 📊 Category-wise expense chart (Bar Graph)
* 📅 Monthly expense tracking (optional feature)

---

## 🛠️ Technologies Used

* Java (Core + OOP)
* Java Swing (GUI)
* AWT Graphics (for chart visualization)
* Collections Framework (ArrayList, HashMap)

---

## ▶️ How to Run

### 1. Navigate to project folder

```
cd "C:\Expense tracker"
```

### 2. Compile

```
javac *.java
```

### 3. Run

```
java ExpenseTrackerGUI
```

---

## 🖥️ Sample Usage

* Enter expense details (e.g., Food, ₹120, 2026-04-25)
* Click **Add Expense**
* View data in table
* Click **Show Chart** to visualize category-wise spending
* Delete any entry if needed

---

## 📊 Output

* Displays expenses in tabular format
* Shows total spending
